# Fuel-Cost-Estimator
# This is my repository which helps to estimate the fuel cost for a trip
**USE OF THIS TOOL**
We use fuel cost estimates for budgeting and planning, such as for a road trip or managing monthly expenses. They help in making informed decisions, like choosing between different routes or vehicles, and in managing expenses for both personal and business use, like a personal vehicle or a fleet of trucks. 
The feature of this project provides the calculation of fuel cost and helps to create a proper cost estimation for a perfect journey . It's objective is also helps to keep a perfect track of money used on fuel.
**Tools Used** : language:python
**steps to run the project** are:
1) user is asked to enter the distance
2) then is asked to enter the mileage
3) and is asked to enter the fuel type
4) and last is asked to enter the fuel price
5) using the formulas of fuel needed and the total cost 
6) the results are given
